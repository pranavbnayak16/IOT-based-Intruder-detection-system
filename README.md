Intruder Detection System Using Face Recognition

An IoT-based security solution designed to detect and identify unauthorized individuals in real-time.
This project leverages face recognition technology and IoT infrastructure to provide an automated, 
smart surveillance mechanism for homes and restricted areas.

Features:

Real-Time Monitoring: Continuous surveillance via camera feed to detect movement and presence.

Face Recognition: Differentiates between authorized personnel and unknown intruders using machine 
                  learning/computer vision models.

Instant Alerts: Triggers notifications or visual alarms (e.g., LEDs, buzzers) when an unauthorized
                person is detected.

IoT Integration: Connected architecture for seamless data logging and remote status updates.


Hardware Requirements:

Microcontroller/Processor: Raspberry Pi (or ESP32-CAM / equivalent IoT module)

Camera Module: Pi Camera or USB Webcam

Indicators: LEDs and Buzzer for local alarm triggers

Miscellaneous: Breadboard, Jumper wires, and Power Supply


Software & Libraries:

Language: Python

Libraries: OpenCV (opencv-python)
